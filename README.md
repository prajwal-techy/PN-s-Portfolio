# 🌐 Personal Portfolio Website (PN's Portfolio)

A modern and responsive personal portfolio website created using HTML, CSS, and a little bit of JavaScript.

This portfolio represents my journey as a beginner Computer Science student who is learning web development and exploring the tech field through hands-on projects.

---

## 🚀 Features

- Responsive Navigation Bar
- Modern Dark UI with Orange Glow Theme
- About Me Section
- Skills Showcase
- Project Showcase Cards
- Social Media Integration
- Contact Section using EmailJS
- Hover Effects and Smooth Styling
- Beginner-Friendly Clean Layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Basic)
- EmailJS

---

## 📌 About This Project

This project was mainly built using manually written HTML and CSS.  
JavaScript is used only for simple functionalities like menu/navbar interaction.

The EmailJS integration was implemented with the help of GitHub Copilot while learning how contact form services work.

The goal of this project was to improve my frontend development skills, practice website structuring, and learn how real portfolio websites are designed.
